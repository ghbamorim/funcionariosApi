# Gerenciador de funcionários

## Api-service
Api para gerenciamento de funcionários
### Instruções:
Instale as dependências:
  `npm install`;  
Inicialize o serviço:  
`npm start`;  
Testes:  
`npx jest`   
#### Swagger
Você pode acessar a interface do Swagger pela rota /docs. Ex.: 
`http://127.0.0.1:3001/docs`
